# BankingApp

## Features
- Authentication
  - Login
  - Register
  - Logout
- Dashboard
  - Account Summary
  - Transaction History
- Transfer

## Tech Stacks
- Networking
  - Retrofit 2
  - ChuckInterceptor as debugger in DEBUG build
- MVVM + Repository Pattern
  - LiveData
  - Coroutine
  - Shared Preference as local data storage (EncryptedSharedPreferences)
- Dependency Injection
  - Dagger Hilt
- Unit Test
  - JUnit 4
  - Mockito
